# Test de Ventas JAPI HUB

App web para reemplazar el uso actual de ScoreApp dentro del ecosistema de JAPI HUB.

Objetivo: crear una herramienta propia de diagnóstico para evaluar qué tan bien una empresa está vendiendo por WhatsApp, capturar leads y entregar resultados personalizados alineados al branding de JAPI HUB.
